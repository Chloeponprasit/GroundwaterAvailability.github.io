# Spatial Analysis of Groundwater Availability using GIS and Hydrogeological Data 🌊🗺️

This project integrates hydrogeological principles with advanced Geospatial Information Systems (GIS) and data science workflows to perform rigorous spatial analysis and support sustainable water resource management.

---

## 🎯 Project Overview & Objectives
The primary goal of this project is to analyze, map, and quantify groundwater availability through spatial data processing and hydrogeological assessment.
* **Groundwater Potential Mapping:** Utilize spatial interpolation and multi-criteria decision analysis to delineate groundwater potential zones.
* **Geospatial Data Integration:** Fuse diverse spatial and temporal datasets to analyze regional aquifer characteristics and water level trends.
* **Data-Driven Evaluation:** Automate geospatial and statistical workflows to evaluate spatial-temporal changes in groundwater storage.

## 🗺️ Study Area & Datasets
* **Study Region:** Eastern Region of Thailand
* **Key Datasets Utilized:**
  * **Groundwater Wells Data:**
  * **Hydrogeological & Lithological Data:** Spatial layers of aquifer geometry, well yields, and specific capacities.
    

## 🛠️ Methodology & Technical Stack
This project leverages modern geospatial tools and data science packages to build reproducible workflows:
* **GIS & Spatial Analysis:** `ArcGIS Pro` / `QGIS` (Geoprocessing, Spatial Interpolation like Kriging/IDW, and Raster Analysis).
* **Programming Languages & Data Science:** `Python` (Geopandas, Pandas, NumPy, Matplotlib, SciPy) for spatial data manipulation and statistical plotting.
* **Version Control:** `Git` for collaborative, organized, and reproducible research.
